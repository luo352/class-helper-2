# 班级助手2.0 - 用户操作指南

## 一、系统介绍

班级助手2.0是一款专为学生设计的课程管理与学习辅助工具，提供课表管理、考勤打卡、路线导航、课程提醒等核心功能。本指南帮助您快速了解并使用系统。

---

## 二、访问方式

### 2.1 通过二维码访问

**步骤：**
1. 打开浏览器访问二维码页面：`http://localhost:10086/qr-code.html`（开发环境）
2. 使用手机扫码工具扫描页面上的二维码
3. 手机将自动跳转到班级助手系统

**二维码页面功能：**
- **复制链接**: 点击"复制链接"按钮，一键复制访问地址
- **保存二维码**: 点击"保存二维码"按钮，将二维码保存为图片文件
- **自动适配**: 二维码页面会自动识别当前部署地址，生成对应的二维码

**保存二维码：**
- 在二维码页面点击"保存二维码"按钮
- 将图片保存到手机相册或电脑中
- 以后可直接扫描保存的图片访问系统

### 2.2 通过链接访问

**开发环境（本地测试）：**
```
http://localhost:10086/
```

**生产环境（正式访问）：**
```
https://your-domain.com/
```

**二维码页面地址：**
- 开发环境：`http://localhost:10086/qr-code.html`
- 生产环境：`https://your-domain.com/qr-code.html`

**保存链接：**
- **iOS Safari**: 点击底部"分享"按钮 → 选择"添加到主屏幕"
- **Android Chrome**: 点击右上角"..." → 选择"添加到主屏幕"
- **桌面浏览器**: 将链接添加到书签或收藏夹

## 三、功能操作

### 3.1 用户登录

**首次登录：**
1. 打开系统后，进入登录页面
2. 输入用户名和密码
3. 点击"登录"按钮
4. 登录成功后自动跳转首页

**登录态管理：**
- 登录后系统自动保存登录状态
- 下次访问自动登录，无需重复输入
- 登录状态有效期为7天

**退出登录：**
1. 进入"我的"页面
2. 点击"退出登录"按钮
3. 确认退出后返回登录页面

### 3.2 课表管理

**查看周视图：**
1. 点击底部"课表"标签
2. 默认显示周视图
3. 可看到一周所有课程安排

**查看日视图：**
1. 在课表页面点击"日视图"切换按钮
2. 显示当天的课程详情
3. 点击日期标签可切换查看其他日期

**周数导航：**
- 点击"上一周"按钮：查看上一周课程
- 点击"下一周"按钮：查看下一周课程
- 点击"回到今天"按钮：快速回到本周视图

**添加课程：**
1. 点击课表页面右下角"+"按钮
2. 填写课程信息（课程名称、教师、教室、时间等）
3. 点击"保存"按钮

**编辑/删除课程：**
1. 在课表中点击要修改的课程
2. 进入课程详情页面
3. 点击"编辑"或"删除"按钮

### 3.3 考勤打卡

**打卡操作：**
1. 点击底部"考勤"标签
2. 点击"立即打卡"按钮
3. 系统自动记录打卡时间和位置
4. 打卡成功后显示打卡记录

**查看记录：**
1. 在考勤页面切换到"打卡记录"标签
2. 可查看历史打卡记录
3. 支持按日期筛选

### 3.4 路线导航

**使用导航：**
1. 在首页或课程详情页面点击"去导航"按钮
2. 系统自动获取当前位置
3. 显示从当前位置到教室的路线
4. 点击"开始导航"按钮跳转到地图应用

**手动输入目的地：**
1. 在导航页面手动输入教室名称
2. 系统自动搜索教室位置
3. 显示路线规划

### 3.5 课程提醒

**开启提醒：**
1. 进入"我的"页面
2. 点击"提醒设置"
3. 开启"课程提醒"开关
4. 设置提前提醒时间（5/10/15/30/60分钟）

**提醒方式：**
- **应用内提醒**: 课程开始前在应用内弹出提醒
- **浏览器通知**: 开启推送通知后，即使关闭页面也能收到提醒

### 3.6 个人中心

**编辑资料：**
1. 进入"我的"页面
2. 点击"个人资料"
3. 修改昵称、头像等信息
4. 点击"保存"按钮

**数据备份：**
1. 进入"我的"页面
2. 点击"数据备份"
3. 点击"导出数据"按钮
4. 将数据保存到本地文件

**数据恢复：**
1. 在数据备份页面点击"导入数据"
2. 选择之前导出的备份文件
3. 确认导入后恢复数据

## 五、常见问题

### Q1: 为什么页面加载缓慢？

**可能原因：**
- 网络连接不稳定
- 浏览器缓存问题
- 设备性能较低

**解决方案：**
1. 检查网络连接
2. 清除浏览器缓存（Ctrl + Shift + Delete）
3. 尝试使用其他浏览器
4. 关闭其他占用资源的应用

### Q2: 定位功能无法使用？

**可能原因：**
- 浏览器未授权定位权限
- 设备定位服务未开启

**解决方案：**
1. 检查浏览器定位权限设置
2. 确保设备定位服务已开启
3. 刷新页面重新授权

### Q3: 课程提醒不生效？

**可能原因：**
- 提醒设置未开启
- 浏览器通知权限未授权

**解决方案：**
1. 进入提醒设置页面检查开关状态
2. 在浏览器设置中开启通知权限
3. 确保应用在后台运行

### Q4: 数据丢失怎么办？

**解决方案：**
1. 定期导出数据备份
2. 导入之前的备份文件恢复数据
3. 联系管理员寻求帮助

### Q5: 如何在手机桌面创建快捷方式？

**iOS：**
1. 使用Safari打开系统
2. 点击底部"分享"按钮
3. 选择"添加到主屏幕"
4. 点击"添加"完成

**Android：**
1. 使用Chrome打开系统
2. 点击右上角"..."菜单
3. 选择"添加到主屏幕"
4. 点击"添加"完成
